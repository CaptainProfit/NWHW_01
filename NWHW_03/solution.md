## **Цель*:
1. Определение подсетей по IPv4-адресу
2. Расчет подсетей по IPv4-адресу
## Проблема 1

| Дано:                       |                 |
| --------------------------- | --------------- |
| **IP-адрес узла:**          | 192.168.200.139 |
| **Исходная маска подсети:** | 255.255.255.0   |
| **Новая маска подсети:**    | 255.255.255.224 |

| Найти:                                          |
| ----------------------------------------------- |
| **Количество бит подсети**                      |
| **Количество созданных подсетей**               |
| **Количество бит узлов в подсети**              |
| **Количество узлов в подсети**                  |
| **Сетевой адрес этой подсети**                  |
| **IPv4-адрес первого узла в этой подсети**      |
| **IPv4-адрес последнего узла в этой подсети**   |
| **Широковещательный IPv4-адрес в этой подсети** |
224 = 128 + 64 +32 = 0xE0 - 3 бита
139 = 128 + 8 + 2 + 1 = 0x8B
**Сетевой адрес этой подсети** это побитовое И новой маски и IP узла = 192.168.200.128
**IPv4-адрес первого узла в этой подсети** - следующий после него адрес 192.168.200.129
**Широковещательный IPv4-адрес в этой подсети** - берем сетевой адрес и делаем максимальным суфикс (5 последних бит единицы) 192.168.200.0x9F = 192.168.200.159
**IPv4-адрес последнего узла в этой подсети** - предыдущий адрес - 192.168.200.158

| Итого:                                          |                                      |
| ----------------------------------------------- | ------------------------------------ |
| **Количество бит подсети**                      | была маска 24, стала 27, значит 3    |
| **Количество созданных подсетей**               | подразбили сеть на 2^3 = 8 подсетей. |
| **Количество бит узлов в подсети**              | =32 - 27 = 5                         |
| **Количество узлов в подсети**                  | 2^5-2 = 30                           |
| **Сетевой адрес этой подсети**                  | 192.168.200.128                      |
| **IPv4-адрес первого узла в этой подсети**      | 192.168.200.129                      |
| **IPv4-адрес последнего узла в этой подсети**   | 192.168.200.158                      |
| **Широковещательный IPv4-адрес в этой подсети** | 192.168.200.159                      |
